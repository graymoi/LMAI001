# 文件夹管理自动化技能

## 技能概述

本技能用于自动化文件夹管理，包括文件整理、文件夹优化、文件夹进化等功能，确保项目文件夹结构清晰、规范、可进化。

---

## 技能功能

### 1. 文件整理功能

#### 1.1 自动分类

**功能描述：** 根据文件类型和内容自动分类文件

**分类规则：**
- 文档文件 → 03_文档和报告
- 代码文件 → 05_代码文件
- 数据文件 → 04_数据文件
- 工具脚本 → 02_工具和脚本
- 临时文件 → 99_临时文件

**执行步骤：**
1. 扫描项目文件夹
2. 识别文件类型
3. 根据规则分类
4. 移动文件到对应文件夹
5. 记录整理日志
6. 生成整理报告

#### 1.2 自动命名

**功能描述：** 根据命名规范自动重命名文件

**命名规则：**
- 文档文件：[日期]-[项目名称]-[文档类型]-[版本].md
- 代码文件：[模块名称]-[功能描述].py
- 配置文件：[配置类型]-[配置名称].json

**执行步骤：**
1. 扫描项目文件夹
2. 识别不符合规范的文件名
3. 根据规则生成新文件名
4. 重命名文件
5. 记录重命名日志
6. 生成重命名报告

#### 1.3 自动去重

**功能描述：** 自动识别并处理重复文件

**去重策略：**
- 基于文件名去重
- 基于文件内容去重
- 基于文件哈希去重

**执行步骤：**
1. 扫描项目文件夹
2. 计算文件哈希值
3. 识别重复文件
4. 标记重复文件
5. 生成去重报告
6. 等待用户确认处理

## 技能使用

### 使用方式

#### 1. 命令行使用

```bash
# 整理文件
python folder_automation.py organize --path=f:\AIlm\000项目管理

# 重命名文件
python folder_automation.py rename --path=f:\AIlm\000项目管理

# 去重文件
python folder_automation.py deduplicate --path=f:\AIlm\000项目管理

# 优化文件夹
python folder_automation.py optimize --path=f:\AIlm\000项目管理

# 监控文件夹
python folder_automation.py monitor --path=f:\AIlm\000项目管理
```

#### 2. API使用

```python
from folder_automation import FolderAutomation

# 创建技能实例
fa = FolderAutomation()

# 整理文件
fa.organize_files(path="f:\\AIlm\\000项目管理")

# 重命名文件
fa.rename_files(path="f:\\AIlm\\000项目管理")

# 去重文件
fa.deduplicate_files(path="f:\\AIlm\\000项目管理")

# 优化文件夹
fa.optimize_folder(path="f:\\AIlm\\000项目管理")

# 监控文件夹
fa.monitor_folder(path="f:\\AIlm\\000项目管理")
```

#### 3. 智能体调用

## 附录

### 常见问题

**Q1: 如何自定义分类规则？**

A: 在配置文件的auto_organize.rules中添加或修改分类规则。

**Q2: 如何自定义命名规则？**

A: 在配置文件的auto_rename.rules中添加或修改命名规则。

**Q3: 如何设置监控告警？**

A: 在配置文件的monitoring中设置监控和告警参数。

**Q4: 如何优化文件夹性能？**

A: 使用optimize功能，选择performance_optimization选项即可优化文件夹性能。
