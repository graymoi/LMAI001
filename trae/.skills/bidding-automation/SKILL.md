# 标书编写自动化技能

## 技能概述

本技能用于自动化标书编写工作流程，包括文档生成、质量检查、格式优化等功能。

---

## 技能功能

### 1. 文档生成功能

#### 1.1 需求规格说明书生成

**功能描述：** 根据需求大纲自动生成需求规格说明书

**输入参数：**
- 需求大纲文档路径
- 项目基本信息
- 技术要求清单

**输出结果：**
- 完整的需求规格说明书文档
- 文档结构符合大纲要求
- 内容完整且准确

**执行步骤：**
1. 读取需求大纲文档
2. 分析大纲结构
3. 根据大纲生成各章节内容
4. 整合内容生成完整文档
5. 检查文档完整性和准确性
6. 输出最终文档

#### 1.2 技术要求文档生成

**功能描述：** 根据招标文件自动生成技术要求文档

**输入参数：**
- 招标文件路径
- 技术要求清单
- 参考文档路径

**输出结果：**
- 完整的技术要求文档
- 技术要求完整且准确
- 符合招标文件要求

**执行步骤：**
1. 读取招标文件
2. 提取技术要求
3. 分析参考文档
4. 生成技术要求文档
5. 检查文档完整性
6. 输出最终文档

#### 1.3 技术方案文档生成

**功能描述：** 根据需求规格说明书自动生成技术方案文档

**输入参数：**
- 需求规格说明书路径
- 技术方案模板
- 技术架构方案

**输出结果：**
- 完整的技术方案文档
- 技术方案完整且可行
- 符合需求规格说明书要求

**执行步骤：**
1. 读取需求规格说明书

## 技能配置

### 配置文件

```json
{
  "skill_name": "标书编写自动化",
  "skill_version": "1.0.0",
  "skill_type": "自动化",
  "skill_category": "文档管理",
  "enabled": true,
  "config": {
    "document_templates": {
      "需求规格说明书": "templates/需求规格说明书.md",
      "技术要求文档": "templates/技术要求文档.md",
      "技术方案文档": "templates/技术方案文档.md"
    },
    "quality_checks": {
      "content_completeness": true,
      "format_standardization": true,
      "technical_accuracy": true
    },
    "optimization_settings": {
      "content_optimization": true,
      "format_optimization": true,
      "structure_optimization": true
    },
    "export_settings": {
      "supported_formats": ["pdf", "docx", "html", "xlsx"],
      "default_format": "pdf",
      "export_path": "output/"
    }
  }
}
```

### 配置说明

| 配置项 | 说明 | 默认值 |
|-------|------|--------|
| skill_name | 技能名称 | 标书编写自动化 |
| skill_version | 技能版本 | 1.0.0 |
| skill_type | 技能类型 | 自动化 |
| skill_category | 技能分类 | 文档管理 |
| enabled | 是否启用 | true |
| document_templates | 文档模板路径 | templates/ |
| quality_checks | 质量检查设置 | 全部启用 |
| optimization_settings | 优化设置 | 全部启用 |
| export_settings | 导出设置 | PDF格式 |

## 技能使用

### 使用方式

#### 1. 命令行使用

```bash
# 生成需求规格说明书
python bidding_automation.py generate --type=需求规格说明书 --input=大纲.md --output=需求规格说明书.md

# 检查文档质量
python bidding_automation.py check --input=需求规格说明书.md --type=质量检查

# 优化文档
python bidding_automation.py optimize --input=需求规格说明书.md --type=内容优化

# 导出文档
python bidding_automation.py export --input=需求规格说明书.md --format=pdf --output=需求规格说明书.pdf
```

#### 2. API使用

```python
from bidding_automation import BiddingAutomation

# 创建技能实例
ba = BiddingAutomation()

# 生成文档
document = ba.generate_document(
    document_type="需求规格说明书",
    input_file="大纲.md",
    output_file="需求规格说明书.md"
)

# 检查文档质量
quality_report = ba.check_document(
    input_file="需求规格说明书.md",
    check_type="质量检查"
)

# 优化文档
optimized_document = ba.optimize_document(
    input_file="需求规格说明书.md",
    optimize_type="内容优化"
)

# 导出文档
exported_file = ba.export_document(
    input_file="需求规格说明书.md",
    export_format="pdf",
    output_file="需求规格说明书.pdf"
)
```

## 附录

### 常见问题

**Q1: 如何自定义文档模板？**

A: 在templates目录下创建新的模板文件，然后在配置文件中添加模板路径。

**Q2: 如何添加新的质量检查规则？**

A: 在quality_checks配置中添加新的检查项，并实现对应的检查逻辑。

**Q3: 如何优化文档生成速度？**

A: 可以通过启用缓存、并行处理、使用模板引擎等方式优化生成速度。

**Q4: 如何导出多种格式的文档？**

A: 使用export功能，指定不同的export_format参数即可导出不同格式的文档。
