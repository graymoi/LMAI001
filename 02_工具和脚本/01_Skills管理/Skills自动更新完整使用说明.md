### 步骤2: 配置自动运行

**方法A: 使用任务计划程序（推荐）**

1. **打开任务计划程序**
   - 按 `Win + R`
   - 输入 `taskschd.msc`

2. **创建任务**
   - 点击"创建基本任务"
   - 名称：`Skills自动检查和更新`
   - 描述：`自动检查并更新npx skills`

3. **设置触发器**
   - 选择"新建触发器"
   - 选择"按计划"
   - 设置：每周 → 周日 → 凌晨2:00

4. **设置操作**
   - 启动程序：浏览到 `F:\AIlm\000项目管理\auto_update_skills.bat`
   - 起始于：`F:\AIlm\000项目管理`

5. **完成**
   - 勾选"当单击完成时，打开此任务属性的对话框"
   - 点击"完成"

6. **高级设置（推荐）**
   - 条件：勾选"只有在计算机使用交流网络连接时才启动此任务"
   - 设置：如果任务运行时间超过30分钟，停止任务

**方法B: 使用XML导入（最快）**

1. **打开任务计划程序**
   - 按 `Win + R`
   - 输入 `taskschd.msc`

2. **导入任务**
   - 右侧点击"导入任务"
   - 选择 `skills_auto_update.xml`
   - 点击"打开"

3. **验证导入**
   - 查看任务列表
   - 确认"Skills自动检查和更新"任务已创建

## 🔧 故障排除

### 问题1: 任务未运行

**检查清单**:
- [ ] 任务是否已启用（不是"已禁用"）
- [ ] 触发器设置是否正确（每周日凌晨2:00）
- [ ] 计算机是否在指定时间开机
- [ ] 脚本路径是否正确

**解决方案**:
1. 手动运行脚本测试：`auto_update_skills.bat`
2. 检查任务计划程序服务是否运行：
   ```bash
   # 以管理员身份运行PowerShell
   Get-Service Schedule
   ```
3. 重启任务计划程序服务：
   ```bash
   # 以管理员身份运行
   net stop Schedule
   net start Schedule
   ```

### 问题2: 脚本执行失败

**检查日志文件**:
1. 打开最新的日志文件
2. 查找错误信息：`[ERROR]`
3. 查找警告信息：`[WARNING]`

**常见错误和解决方案**:

**错误1**: `npx未找到`
```
[错误] npx未找到，请先安装Node.js和npm
```
**解决方案**:
- 访问 https://nodejs.org/
- 下载并安装Node.js LTS版本
- 安装后，npx会自动可用

**错误2**: `检查过程中出现错误`
```
[警告] 检查过程中出现错误，代码: 1
```
**解决方案**:
- 检查网络连接
- 检查npx skills是否正确安装
- 手动运行：`npx skills check`

**错误3**: `更新过程中出现错误`
```
[错误] 更新过程中出现错误，代码: 1
```
**解决方案**:
- 检查磁盘空间
- 检查文件权限
- 手动运行：`npx skills update`

### 问题3: 网络连接问题

**症状**:
- 任务显示"正在运行"但长时间不完成
- 日志显示网络错误

**解决方案**:
1. 确保任务设置中勾选了"只有在计算机使用交流网络连接时才启动此任务"
2. 检查防火墙设置
3. 增加超时时间：将执行时间限制从30分钟增加到60分钟

## 🎯 高级配置

### 修改更新频率

**更频繁的检查（每天）**:
1. 打开任务属性
2. 修改触发器：每天 → 凌晨2:00
3. 保存设置

**更少的检查（每月）**:
1. 打开任务属性
2. 修改触发器：每月 → 第1个周日 → 凌晨2:00
3. 保存设置

### 添加邮件通知

**配置SMTP服务器**:
1. 打开任务属性
2. 切换到"操作"选项卡
3. 点击"新建" → "发送电子邮件"
4. 配置：
   - SMTP服务器：`smtp.gmail.com`（示例）
   - 发件人：`your-email@gmail.com`
   - 收件人：`your-email@gmail.com`
   - 主题：`Skills更新完成通知`
   - 服务器要求身份验证：是
   - 用户名：`your-email@gmail.com`
   - 密码：`your-app-password`

### 添加桌面通知

**配置显示消息**:
1. 打开任务属性
2. 切换到"操作"选项卡
3. 点击"新建" → "显示消息"
4. 配置：
   - 标题：`Skills更新完成`
